# M3 问题定型 — 输出示例

## 技术矛盾示例

### 矛盾类型

技术矛盾

### 矛盾对

- **改善方面**：结构强度
- **恶化方面**：建造成本

### 矛盾描述

提高建筑物抗震能力需要增强结构强度，但会导致建造成本大幅上升。

### 依据

根因参数指向"结构强度"，关键问题为抗震能力不足；改善结构强度会直接增加材料用量和施工难度，从而推高建造成本。

## 物理矛盾示例

### 矛盾类型

物理矛盾

### 矛盾对

- **改善方面**：大容量
- **恶化方面**：轻薄

### 矛盾描述

既要电池容量大又要手机轻薄。

### 依据

用户需求是"续航长"（需要大电池）与"便携"（需要轻薄手机）的对立；这是同一参数的两个对立状态，属于物理矛盾，应使用分离原理解决。
