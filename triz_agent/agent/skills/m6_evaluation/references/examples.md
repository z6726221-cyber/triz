# M6 方案评估 — 输出示例

## 示例 1：手机电池续航问题（3 个方案评估）

**用户原始问题**："手机电池续航太短，一天要充两次电，怎么办？"

### 方案评估

#### 方案 1：石墨烯电池替换方案

| 维度 | 评分 |
|------|------|
| 技术可实现性 | 3/5 |
| 资源匹配度 | 2/5 |
| 创新性 | 4/5 |
| 独特性 | 4/5 |
| 风险等级 | high |
| 问题匹配度 | 3/5 |
| 逻辑一致性 | 3/5 |
| IFR偏离原因 | IFR 要求"无需用户干预"，但此方案仍需用户主动更换电池 |

**评分依据**：石墨烯电池技术尚未成熟，量产成本高（资源匹配度低）；更换电池需用户干预，偏离 IFR；技术可行性中等，风险较高。

#### 方案 2：自适应后台进程管理方案

| 维度 | 评分 |
|------|------|
| 技术可实现性 | 5/5 |
| 资源匹配度 | 5/5 |
| 创新性 | 3/5 |
| 独特性 | 3/5 |
| 风险等级 | low |
| 问题匹配度 | 5/5 |
| 逻辑一致性 | 5/5 |
| IFR偏离原因 | （无） |

**评分依据**：完全利用现有资源（后台进程管理、屏幕亮度自适应），无需新增硬件；用户无需干预，系统自动管理功耗；技术成熟，风险低；与用户原始问题"一天充两次电"高度匹配。

#### 方案 3：硅纳米线快充电池方案

| 维度 | 评分 |
|------|------|
| 技术可实现性 | 4/5 |
| 资源匹配度 | 4/5 |
| 创新性 | 5/5 |
| 独特性 | 5/5 |
| 风险等级 | medium |
| 问题匹配度 | 4/5 |
| 逻辑一致性 | 4/5 |
| IFR偏离原因 | （无） |

**评分依据**：硅纳米线电池能量密度高，充电快，利用现有充电电路资源；技术较新但已有商用案例；问题匹配度好，风险中等。
