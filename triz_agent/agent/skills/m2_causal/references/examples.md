# M2 根因分析 — 输出示例

## 示例 1：手机电池续航问题（基于 m1 输出）

**基于 m1 输出**：电池有负面功能"电池 消耗 电能（自放电）"→ harmful，"用户 充电 电池"→ excessive

### 根因参数

电能转化率

### 关键问题

电池在低电量状态下充放电效率下降，导致可用容量衰减，用户需频繁充电

### 因果链

1. 第1层：电池续航短 → 用户需一天充两次电
2. 第2层：为什么续航短？→ 电池自放电率高，待机时也消耗电能
3. 第3层：为什么自放电高？→ 电解液阻抗随电量降低而升高，充放电转化率下降
4. 第4层（根因）：电能-化学能转换效率低，低电量时转化效率进一步恶化

### 候选物理属性

- 电能转化率
- 电解液阻抗
- 化学能密度
- 充放电效率
